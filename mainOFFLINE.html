Help:

If timer is running, led is red and button shows off, it means that the change on microcontroller will be made when timer stops running. It shows the change that is going to happen after timer stops running.

Worker_DHT22_x:
logData -> adds another log to existing dataLog.json file //dataLog("logData")
checkDataLog -> checks if dataLog.json file is written correctly and if not, it resets file //dataLog("checkDataLog")
resetDataLog -> resets dataLog.json file //dataLog("resetDataLog")
updateDHTFile -> reads temperature and humidity and saves it into DHT22.json file //updateDHTFile()
intervalDHT:x -> changes intervalDHT variable, function updateDHTFile() is now called every x seconds (3s min, 255s max)
intervalDataLog:x -> changes intervalDataLog variable, function dataLog("logData") is now called every x hours
sendIntervals -> sends millis for each interval variable
